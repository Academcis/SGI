	2º Carregar no botão da lupa na barra de pesquisa 
	3º Carregar na pesquisa recente "móvel de jardinagem" 
	4º Dá para ver os diferentes filtros que idealmente se implementaria 
	5º Fazer hover na "Mesa de jardinagem, em acácia" 
	6º Carregar no carrinho

	9º Scroll para baixo na pág., dá para ver a forma como mudávamos a disposição das 'Especificações do Produto'

	13º Carregar no carrinho para ver as alterações
