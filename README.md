<h1>Project Description:</h1>
<p>The project provides an example of how we would solve usability and interaction issues identified on LaRedoute's website. Since the course was at an early stage, the implementation consists of a static HTML website with linked pages that allow the user to follow the product flow.</p>

<h1>How to Explore the Project (Figma Flow):</h1>
<ol>
<li>Click 'Present' in Figma (top right) and select "Flow 5".</li>
<li>Use the search bar to look for "m√≥vel de jardinagem".</li>
<li>Observe the filters and product hover interactions.</li>
<li>Click on products to view details and add them to the cart.</li>
<li>Use product comparison and navigate between product pages.</li>
<li>Explore the interactive 3D models on the product pages to open/close doors and view animations.</li>
</ol>

<h1>3D Interaction Details (Three.js):</h1>
<ul>
<li>Products "M√≥vel Jardinagem" and "Mesa Jardinagem" allow opening/closing doors by clicking directly or via buttons.</li>
<li>Clicking the LaRedoute logo redirects to the product list page.</li>
<li>3D interactions are accessible from <code>lista_produtos.html</code>, showcasing the two implemented products.</li>
</ul>

<h1>Tools Used</h1>
<ul>
<li>HTML/CSS/JavaScript: For the static website structure and flow.</li>
<li>Three.js: For 3D interactive models.</li>
<li>Blender: For creating 3D animations of the product models.</li>
<li>Figma: For prototyping and designing the product flow.</li>
</ul>
